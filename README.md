# FitEliteCO Coach - ZIP completo final verificado

App Expo + React Native Web + Supabase.

## Configuración Vercel

- Framework Preset: Other
- Root Directory: ./
- Install Command: npm install --legacy-peer-deps
- Build Command: npm run build:web
- Output Directory: dist

## Variables Vercel

- EXPO_PUBLIC_SUPABASE_URL
- EXPO_PUBLIC_SUPABASE_ANON_KEY

## Supabase

Ejecuta el archivo:

supabase/schema.sql

en Supabase > SQL Editor.

## Nota importante

Este ZIP no incluye archivos PNG ni referencias a favicon/icon/splash para evitar el error CRC de Jimp en Vercel.
